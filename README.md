# Componente-practico-1
Implementar algoritmos de manipulación de vectores numéricos utilizando JavaScript, agregando funcionalidades específicas al sistema base proporcionado.
Manipulación de Vectores

# Introducción
En este componente se desarrollará una serie de ejercicios prácticos enfocados en la manipulación de vectores (arreglos unidimensionales). El sistema permitirá al usuario definir las dimensiones del arreglo y realizar búsquedas de valores específicos. Interfazmente, incluirá una tabla dinámica con controles para cargar y vaciar los elementos del vector.

Asimismo, se incorporarán funcionalidades para desplegar la totalidad de los datos, identificar los valores máximo y mínimo, y calcular tanto la sumatoria como el producto de los elementos. En el ámbito estadístico, el sistema computará la media, la mediana y la moda, visualizando los resultados directamente en la interfaz. Finalmente, se implementarán algoritmos de ordenamiento:

Ordenamiento Burbuja (ASC/DESC): Compara elementos adyacentes e intercambia sus posiciones de manera iterativa (llevando los valores al final o al inicio según el orden requerido).

Ordenamiento por Selección (ASC/DESC): Busca activamente el elemento menor (o mayor) de la sección no ordenada y lo intercambia con el elemento de la posición inicial correspondiente, avanzando progresivamente desde el principio hacia el final del arreglo.

# Objetivo
Implementar algoritmos de manipulación de vectores numéricos utilizando JavaScript, agregando funcionalidades específicas al sistema base proporcionado.

# Documentación
El sistema permite manipular un vector numérico representado en una tabla HTML mediante JavaScript. Se pueden cargar valores aleatorios, vaciar el vector o mostrar su contenido al usuario. Incluye operaciones básicas como el cálculo del número mayor, menor, suma y producto. También implementa medidas estadísticas como media, mediana y moda. Los resultados se muestran dinámicamente en la sección de resultados del documento. Además, permite ordenar el vector usando los métodos de burbuja y selección en orden ascendente o descendente. Este sistema sirve para practicar la manipulación del DOM, el manejo de arreglos y la lógica de algoritmos básicos en JavaScript.

# Conclusión
En conclusión, el desarrollo de este componente facilitó la comprensión y el aprendizaje de diversas metodologías para implementar múltiples funcionalidades dentro de una misma aplicación. Esto incluye el diseño de tablas numéricas dinámicas, estructuras de datos anidadas y la vinculación de eventos a distintos botones de acción. El aspecto más significativo fue la correcta asimilación teórica y práctica de los algoritmos de ordenamiento.
